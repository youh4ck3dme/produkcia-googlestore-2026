## 🚀 Ďalšie Kroky

### 1. Reštartuj Cursor
```bash
# Zatvor Cursor úplne (Cmd+Q)
# A znova ho otvor
```

### 2. Otvor PDF Súbor
- Klikni na `bizagentvssumuop.pdf` v Exploreri
- PDF by sa mal automaticky otvoriť v preview móde

### 3. Ak sa neotvorí automaticky
- Klikni pravým tlačidlom na PDF súbor
- Vyber **"Open With..."**
- Vyber **"PDF Preview"**

## ⌨️ Klávesové Skratky

- **Zoom In:** `Cmd +` (macOS) / `Ctrl +` (Windows/Linux)
- **Zoom Out:** `Cmd -` / `Ctrl -`
- **Fit to Width:** `Cmd 0` / `Ctrl 0`
- **Fit to Page:** `Cmd 1` / `Ctrl 1`
- **Next Page:** `Cmd →` / `Ctrl →` alebo `Page Down`
- **Previous Page:** `Cmd ←` / `Ctrl ←` alebo `Page Up`
